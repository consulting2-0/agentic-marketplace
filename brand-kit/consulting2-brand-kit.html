06 — Brand Pillars
Four Pillars
These four pillars define every product decision, content piece, and partnership.
Agentic First
Every feature is built for autonomous agents — not just tools wrapped in AI. Orchestration, skills, hooks, and MCP are first-class citizens.
Radically Open
Every agent, skill, and integration is open source. The marketplace thrives on community contributions. Transparency is non-negotiable.
Enterprise Ready
Deep SAP BTP integration from day one. Built by practitioners who've shipped real enterprise projects. Production patterns, not demos.
Education Embedded
Every listing links to learning. Courses, workshops, and certifications are built into the marketplace — not bolted on as an afterthought.

08 — Usage Rules
Do & Don't
Protect the brand by following these foundational rules.
✓ Do
Use the full wordmark on all primary marketing
Maintain clear space of at least 1× the icon height
Use only approved color combinations
Keep the hexagon proportions locked
Use Syne for all headlines
Use JetBrains Mono for all code and technical labels
Keep the mint node in the logo always
Scale the icon proportionally only
✗ Don't
Stretch, rotate, or skew the logo
Use the logo on busy photographic backgrounds
Change the typeface in the wordmark
Add drop shadows or outer glows to the icon
Use logo colors on unapproved backgrounds
Recreate the icon from scratch
Use "v2.0" or "Version 2.0" as alternatives
Mix brand colors with generic grays for UI
